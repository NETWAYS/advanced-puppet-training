# Adding Functionality 

* Functions are evaluated during compilation
* Functions are evaluated on the Master
* Commonly used for:
 * Interfacing with external tools
 * Expand additional functionality to the Puppet DSL

* Example functions:
 * `include`
 * `template`
 * `create_resources`
 * `str2bool`
